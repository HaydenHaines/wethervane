# US Senate Elections — Knowledge Base

## Structure

The US Senate has 100 members — 2 per state, each serving 6-year terms. Terms are staggered into 3 classes so roughly one-third of the Senate is up for election every 2 years.

### Senate Classes

Each state has one senator in one class and the other in a different class — never both in the same class.

### Election Cycle Pattern

| Year | Classes Up | Type |
|------|-----------|------|
| 2024 | I | Presidential |
| 2026 | **II** | **Midterm** |
| 2028 | III | Presidential |
| 2030 | I | Midterm |
| 2032 | II | Presidential |
| 2034 | III | Midterm |

Special elections can occur in any year when a senator dies, resigns, or is removed from office. The governor typically appoints a temporary replacement, and the special election is held at the next general election.

## General Senate Election Mechanics

- **Primary elections**: Most states hold primaries Jun-Sep of election year. Some states use ranked-choice (AK, ME for general).
- **Runoffs**: Georgia requires a runoff if no candidate gets >50%. Louisiana uses jungle primary (all candidates, top 2 advance if no majority).
- **Special elections**: Appointed senators serve until a special election. Timing varies by state law.
- **Midterm dynamics**: The president's party typically loses seats in midterm elections. With a Republican president (Trump) in 2026, historical pattern favors Democrats.
- **Current Senate**: 53R-47D. Democrats need net +4 for majority (need 51 since VP is Republican).
