# Model Versions

See docs/ROADMAP.md for the model versioning scheme.

Roles:
- `current/` -> symlink to active model
- `previous/` -> symlink to last model before current
- `previous_gen_best/` -> symlink to best pre-refactor model
- `county_baseline/` -> frozen 3-cycle baseline (reference point)
- `versions/` -> all historical snapshots
